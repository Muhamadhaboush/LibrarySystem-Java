# Library Management System (Java)

This project is a simple Java console-based library application.  
Books are stored in a text file and the system supports adding, listing, searching, and deleting records.

## Features
- Add new books
- Display all books
- Search books (case-insensitive)
- Delete by index
- Delete by exact book name
- Automatic file creation (books.txt)

## Technologies
- Java
- File I/O (BufferedReader, BufferedWriter)
- Console-based menu system

## How to Run
1. Clone the repository
2. Open the project in any Java IDE (Eclipse, IntelliJ, VS Code)
3. Run the `project.java` file from `src/week2_thrusday/`

## File Structure
- `books.txt` — stores all book records
- `project.java` — main application file
